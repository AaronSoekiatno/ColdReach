# ColdReach Journey-Focused UI Redesign Plan

## Overview
Transform the website to emphasize the complete ColdReach journey with a "YOU"-focused narrative that highlights the three core value propositions:
1. **Smart Matching** - Match YOU with companies that align with YOUR experiences
2. **AI-Crafted Messages** - Craft unique, compelling cold DMs tailored for YOU
3. **Automated Outreach** - Save YOU weeks by automating professional outreach

## Current State Analysis

### Existing Features
✅ **Resume upload & AI parsing** - Extract skills, education, projects, internships
✅ **Vector-based matching** - Find startups using semantic similarity (Pinecone + Gemini embeddings)
✅ **Match display** - Show matched startups with scores on /matches page
✅ **Email generation** - AI-powered cold email creation using Gemini 2.5 Pro (`lib/email-generation.ts`)
✅ **Email sending** - Gmail OAuth integration to send emails (`SendEmailButton` component)
✅ **Features section** - Basic 4-card feature grid

### Current User Flow
1. User uploads resume → AI extracts data
2. System finds matching startups → Shows results modal with count
3. User navigates to /matches → See all matched startups
4. User can click "Send Email" → Generates & sends cold email via Gmail

### Gaps to Address
❌ Journey is not clearly communicated on landing page
❌ No visual step-by-step process shown
❌ Features section doesn't emphasize the 3-step journey
❌ Hero copy doesn't focus on "YOU" and time-saving benefits
❌ No preview/showcase of AI-generated messages
❌ Missing emphasis on automation and time savings (weeks → minutes)

5. **Tone**: How casual vs. professional should the "YOU"-focused copy be? Current tone is professional-casual.
